# @stratamu/interfaces

This package defines TypeScript interfaces for different game types supported by the Stratamu engine:
- MUD
- MUSH
- MUCK
- MOO

## Usage
Implement these interfaces in your game logic or plugins to extend the core engine for specific game types.

## Scripts
- `bun run dev` — Start development server
- `bun run build` — Build the package
- `bun run test` — Run tests

## TypeScript
- Inherits shared config from the monorepo root
