# @stratamu/plugins

This package contains plugins and integrations for the Stratamu engine, such as database connectors and external package support.

## Example Plugins
- MongoDB integration
- SQLite integration
- External package integration

## Usage
Add or implement plugins here to extend the functionality of your game engine.

## Scripts
- `bun run dev` — Start development server
- `bun run build` — Build the package
- `bun run test` — Run tests

## TypeScript
- Inherits shared config from the monorepo root
