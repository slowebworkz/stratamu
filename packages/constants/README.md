# @stratamu/constants

This package provides shared constants and types for the Stratamu project.

## Usage
Import constants and types from this package in other Stratamu packages.

## Scripts
- `bun run dev` — Start development server
- `bun run build` — Build the package
- `bun run test` — Run tests

## TypeScript
- Inherits shared config from the monorepo root
